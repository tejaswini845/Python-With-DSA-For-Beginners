## ***What is Python?***

Python is a **<mark>Open Source</mark>**<mark>, </mark> **<mark>high-level, interpreted programming language</mark>** created by **<mark>Guido van Rossum</mark>** and released in **<mark>1991</mark>**.

## ***Understanding Machine Level Language and High Level Language***

### **What is Machine Level Language?**

* **It** is the **only language the computer understands directly**.
    
* It is written in **<mark>binary</mark>**(0s and 1s).
    
    Example - 10101010 00010101
    
* Computers (CPU) work using **electrical signals**, so they understand only this form.
    

**Key points:**

1. Fastest for the computer
    
2. Very difficult for humans
    
3. Error-prone
    
4. Hardware dependent
    

Humans **cannot comfortably write** machine-level programs.

### **What is High Level Language?**

* **High-level languages** are designed for humans.
    
* You don’t need to worry about memory management.
    
* The syntax is close to plain English.
    
* Examples - C, C++, Java, Python, JavaScript, etc.
    

**Code in Python -** print("Hello World")

However, the computer **<mark>cannot understand high-level language directly</mark>**. So we need a **<mark>translator</mark>**.

### **What is an <mark>Interpreter</mark>?**

An **interpreter** is a program that acts as a **translator** between <mark>humans</mark> and <mark>computers</mark>.

It :-

* Converts high-level code into machine-level code
    
* Executes the code **line by line**
    
* Shows errors immediately
    

Python uses an **interpreter**.

![](https://cdn.hashnode.com/res/hashnode/image/upload/v1766864624607/3e69eb89-7b2a-4bf8-a573-22b1a4d743fc.png align="center")

### Step-by-step:

1. You write Python code
    
2. Interpreter reads **one line**
    
3. Converts it to machine code
    
4. Executes it immediately
    
5. Moves to the next line
    

### **Why Python Uses an Interpreter?**

**Because:**

* Easy debugging
    
* Beginner friendly
    
* Faster development
    
* Platform independent
    

That’s why Python is **perfect for learning DSA**
