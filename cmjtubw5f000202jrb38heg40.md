## ***<mark>How to Install Python and Set Up Your Coding Environment (Step-by-Step)</mark>***

### Before we start writing Python programs, we need to ***<mark>install Python</mark>*** and set up a **proper *<mark>coding environment</mark>*** on our system.

## **Step 1: Download Python**

1. Open your browser and go to the official Python website:  
    üëâ [<mark>https://www.python.org</mark>](https://www.python.org)
    
2. Click on **Download Python**
    
    The website automatically suggests the latest version for your operating system (Windows / macOS / Linux).
    
3. Download the installer.
    

## **Step 2: Install Python (Important Step)**

1. Open the downloaded installer.
    
2. **VERY IMPORTANT**:  
    ‚úî Check the box **‚ÄúAdd Python to PATH‚Äù**
    
3. Click **Install Now**.
    
4. Wait for the installation to complete.
    

Once finished, click **Close**.

## **Step 3: Verify Python Installation**

### On Windows:

1. Open **Command Prompt**
    
2. Type:
    
    python --version
    
3. Press Enter
    

### On macOS / Linux:

1. Open **Terminal**
    
2. Type:
    
    python3 --version
    

If Python is installed correctly, you will see the Python version number.

## **Step 4: Install <mark>VS Code (Code Editor)</mark>**

VS Code (Visual Studio Code) is a popular and beginner-friendly code editor.

1. Go to üëâ [https://code.visualstudio.com](https://code.visualstudio.com)
    
2. Download VS Code for your operating system
    
3. Install it by following the setup steps
    

## **Step 5: Install <mark>Python Extension</mark> in VS Code**

1. Open **VS Code**
    
2. Click on the **Extensions** icon on the left sidebar
    
3. Search for **Python**
    
4. Install the extension published by **Microsoft**
    

**This extension helps with:**

## **Step 6: Write Your First Python Program**

* Open VS Code
    
* Create a new file named:
    
    [hello.py](http://hello.py)
    
* Write the following code:
    
    print("Hello, Python!")
    

## **Step 7: Run Your Python Program**

* Open Terminal in VS Code
    
* Type:
    
    python [hello.py](http://hello.py)
    
* Press Enter
    
    **Output :**
    
    Hello, Python!
    

üéâ Congratulations! You have successfully run your first Python program.

## Common Mistakes Beginners Make

* Forgetting to check **Add Python to PATH**
    
* Installing Python but not verifying version
    
* Using wrong command (`python` vs `python3`)
    
* Not saving file with `.py` extension
    

## Conclusion

Setting up Python correctly is the **first and most important step** in your programming journey.  
Once your environment is ready, learning Python becomes much easier.
